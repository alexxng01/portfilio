## Troubleshooting

**Servers won't start?**
```bash
# Kill any processes on these ports
lsof -ti:5001 | xargs kill -9    # Backend port
lsof -ti:3000 | xargs kill -9    # Frontend port

# Try again
npm run dev
```

**Database error?**
```bash
# Delete the database (it will recreate with default data)
rm server/portfolio.db

# Restart
npm run dev
```

**Modules missing?**
```bash
# Reinstall
npm run install-all
npm run dev
```
