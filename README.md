# Portfolio CMS - MERN Stack

A complete Content Management System for portfolio websites built with React, Node.js, Express, and MongoDB.

## Features

- **Full Portfolio Management**: Manage profile, about, services, skills, projects, and teamwork sections
- **Admin Dashboard**: Complete admin panel with analytics
- **Contact Form**: Visitors can send messages directly to admin
- **Visitor Tracking**: Track visitors by location, device, and browser
- **Responsive Design**: Works on all devices
- **JWT Authentication**: Secure admin authentication
- **MongoDB Atlas**: Cloud database storage
- **Real-time Updates**: Changes reflect immediately

## Tech Stack

### Frontend
- React 18
- React Router DOM
- Axios
- React Hot Toast
- Recharts (for analytics)
- React Icons
- CSS3 (custom styling)

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- Bcryptjs for password hashing
- Express Validator
