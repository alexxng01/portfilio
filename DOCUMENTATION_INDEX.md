## 📞 Support & Troubleshooting

### Common Issues

**Q: Changes not showing on home page?**
A: Refresh browser (Cmd+R on Mac, Ctrl+R on Windows)

**Q: Admin panel shows 403 error?**
A: Make sure adminToken is in localStorage, login again

**Q: Database not saving?**
A: Check server logs, ensure /server/portfolio.db exists

**Q: Colors look different?**
A: Check browser color settings, try clearing cache

### Getting Help

1. Check `QUICK_REFERENCE_CARD.md` FAQ section
2. Read relevant guide from above list
3. Check `SYSTEM_ARCHITECTURE_DIAGRAM.md` for technical details
4. Review code comments in source files
5. Check browser console for errors
