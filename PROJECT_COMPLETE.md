## 🐛 If Something Goes Wrong

1. **Check TROUBLESHOOTING.md** - Most issues are documented
2. **Clear npm cache** - `npm cache clean --force`
3. **Reinstall dependencies** - `npm run install-all`
4. **Reset database** - `rm server/portfolio.db`
5. **Restart everything** - Stop and run `npm run dev` again
